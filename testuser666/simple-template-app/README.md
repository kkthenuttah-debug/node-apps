# simple-template-app

Testing simplified template copying

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Technologies Used

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Deployment

This project is automatically deployed via the mini-do-backend system.
Access it at: http://localhost:8081/testuser666-simple-template-app/
